## Getting Started

Each project lives in its own folder. To run a project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Healy96/JavaScript-Projects.git
   ```
2. Navigate to the project folder you want to explore:
   ```bash
   cd JavaScript-Projects/<project-name>
   ```
3. Open `index.html` in your browser — no build step required.
